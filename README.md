# 🐳 clase-docker

Repositorio de apuntes y ejemplos prácticos del curso de Docker.  
Cubre desde los fundamentos del sistema (namespaces, cgroups, chroot) hasta orquestación con Docker Swarm.

---

## 📁 Estructura del repositorio

```
clase-docker/
├── 01-fundamentos-namespace-cgroups/   # Teoría y ejemplos en Go
├── 02-comandos-docker/                 # Referencia de comandos esenciales
├── 03-dockerfile/                      # Sintaxis y buenas prácticas
├── 04-docker-compose/                  # Orquestación multi-contenedor
└── 05-docker-swarm/                    # Clústeres y alta disponibilidad
```

---

## 🗂️ Contenido

| Módulo | Descripción |
|--------|-------------|
| [01 - Fundamentos](./01-fundamentos-namespace-cgroups/) | Namespaces, cgroups y chroot desde cero con Go |
| [02 - Comandos Docker](./02-comandos-docker/) | Gestión de contenedores, imágenes, volúmenes y redes |
| [03 - Dockerfile](./03-dockerfile/) | Crear imágenes propias |
| [04 - Docker Compose](./04-docker-compose/) | Escenarios multi-servicio con YAML |
| [05 - Docker Swarm](./05-docker-swarm/) | Clúster de nodos con alta disponibilidad |
